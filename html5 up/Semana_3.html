Definiciones Generales

Un sitio web es un espacio digital en Internet que contiene textos, multimedia o brinda algun servicio, además tiene una dirección y cumple una función en el universo digital. Éstos sitios estan compuestos por diversas paginas web organizadas.

Una página web es un documento que puede estar creado con código HTML, tiene una dirección llamada dominio (.com, .net, .org) y guarda sus datos en un hosting.

Definiciones del HTML

HTML son las siglas de Hypertext Markup Language creado por Tim Berners-Lee en 1991. Está compuesta en dos partes: Cabecera y Cuerpo denominadas HEAD & BODY.

En cuanto a su estructura, el HEAD contiene información básica para el navegador y el BODY contiene títulos, enlaces e imágenes.

Definiciones del CCS

CSS son las siglas de Cascading Style Sheets y es el código que da el estilo al sitio web. Está organizado en reglas y propiedades que define a los elementos en el cual se aplica el estilo.

Una propiedad debe estar acompañado de un valor, como al indicar un color específico a través de una declaración.

¿Qué es un editor?

Es un programa que permite crear y modificar archivos digitales basados en texto para generar instrucciones, realizar configuraciones en un sistema o crear otros programas.

Tenemos una gran variedad de editores, desde el más conocido Block de notas en Windows hasta programas más complejos como Visual Studio Code.
